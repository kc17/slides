### How to search for a sublime package * *Ctrl+Shift+P* is used to show the Command Palette * Start typing: `Package Control` * Choose `Package Control: Install Package` * Here you get to search amoung the tons of packages available [Package Control Usage Ref](https://packagecontrol.io/docs/usage)
