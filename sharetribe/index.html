###What is Sharetribe? Sharetribe is an open source platform to create your own peer-to-peer marketplace.
###Why is Sharetribe? * A sustainable peer econo Foster my where resources are shared and re-used. * Build a meaningful workplace that sets an example for other organizations. * Democratize platform ownership by making it easy for anyone to build and run their own marketplace.
### Alternatives for Sharetribe Practical guides for building an online marketplace business - Airbnb - Etsy - Lyft
###In what ways is this project unique and creative? * Openness * Transparency
### What is the social value of this project? #### The true value of Sharetribe lies in * Real-life encounters. * Sense of empowerment.
### What is the potential of this project to expand and develop? * A cloud-based architecture that makes sure that the platform easily scaled. * A local sharing community with lots of volunteer translators.
### What is the business model of this project? * Work together with local marketplace starters & ambassadors. * Each new tribe has someone who is in charge of it.
### Features of Sharetribe * Global payments made simple. * Selling and renting made easy. * Safe & secure. * Simple & effective. * Trust and reputation
### How to setup sharetribe? * https://github.com/sharetribe/sharetribe
### Pros and cons for building with a Sharetribe backend Pros * Get a fully functional marketplace built in one day. * Need not to write any new code to create a marketplace website * There is a community of other developers building on top of Sharetribe codebase.
### pros and cons for building with a Sharetribe backend Cons * Sharetribe is a tool that is useful for certain purposes, but using it for others might do more harm than good. * As a developer, it's often more fun to start from scratch than to learn an entire complex platform and start extending it. * There might be lots of features that you actually don't want to have, which might be a distraction. If you build your own thing, you get to only build the features you want, which makes development easier, as you avoid complexity.
